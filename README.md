Step 1: Create a New Codespace

Open Your Repository:
Go to your GitHub repository where you want to set up the bot.
Navigate to Codespaces:
Click on the Code tab.
Click on the Codespaces tab.
Create a New Codespace:
Click on the New codespace button.
Select a suitable machine type (e.g., Standard or Large) and click Create codespace.
Step 2: Open a Terminal in Your Codespace

Open the Terminal:
Once your codespace is up and running, you will see a terminal tab at the bottom of the screen.
Click on the terminal tab to open it.
